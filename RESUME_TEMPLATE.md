# 智能新能源车服 AI 管家 (Smart EV Service AI Assistant)

**技术栈：** Python、LangChain、ReAct Agent、ChromaDB、FastAPI、SSE (Server-Sent Events)、HTML5/CSS3、Qwen (通义千问)

**项目描述：** 
基于 Agentic RAG 架构打造的新能源汽车专属智能管家，深度融合车辆遥测数据与垂直领域知识库。实现从自然语言意图识别、多工具动态规划调用到生成拟人化车况分析报告的全自动闭环，为车主日常用车及售后问题排查提供精准、高效的 AI 陪伴式解答能力。

- **基于 LangChain 编排 ReAct Agent 多工具链协作**：设计并打通「思考-行动-观察」链路，底层封装了垂直知识库检索、实时天气预估、VIN 车架号上下文绑定等 7 核心工具节点。赋予智能体动态拆解任务的能力，实现诸如“极端气候预警关联电池损耗”、“个性化驾驶偏好诊断”等多意图复杂场景的自主应对。
- **构建垂直领域专属 RAG 与增量构建机制**：针对新能源“三电”理论与故障盲操，清洗并向量化专业文档至 ChromaDB 数据库。底层开发基于 MD5 哈希校验的文件去重过滤体系，实现数据轻量化增量更新，从根本上消除了通用大模型在汽车专业工程领域的知识“幻觉”。
- **基于 FastAPI 与 SSE 的异步流式交互中枢**：摒弃传统 RESTful 阻塞模式，搭建高性能异步 Streaming 服务端推送链路。在底层状态机精准捕获并清洗 Agent 中间态 Tool Calls 的“碎碎念”思考过程，向前端推送极度纯净且顺滑的逐字流。
- **开发车辆真实物理规律数据引擎**：为了支撑场景化的高级询问，利用 Python 数据清洗与扩展脚本，基于气温衰耗曲线与 SOH 物理规律，自动化构建涵盖全年多季节、多车辆 (VIN1001-1010) 的 100+ 条 CSV 沙盒运行档案，为智能报表生成提供严谨数据闭环。
- **原生高算力级拟物前端架构与状态隔离**：独立全栈实现前台，采用原生 HTML/CSS/JS 构筑极具科技感的高级暗黑毛玻璃 (Premium Glassmorphism) 界面。巧妙运用 JS 作用域机制隐式劫持全局 VIN 查询上下文，杜绝用户侧感知；深度梳理 Flex 响应式缩放边界，攻克频繁流式重绘导致的布局跳闪溃缩难题。
